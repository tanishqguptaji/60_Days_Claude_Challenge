# Day 33 – Media Integrity Analyzer

## ABTalks 60 Days Claude Challenge

## Project Overview

Today I built **Media Integrity Analyzer**, an interactive educational web application designed to improve media literacy through guided learning and real-world style scenarios.

Instead of testing prior knowledge, the application teaches users how to critically evaluate headlines, identify emotional manipulation, assess source reliability, and recognize audience targeting techniques commonly used in digital media.

The simulator provides a premium editorial-style experience where users interact with fictional news articles and social media posts before revealing expert analysis. Every challenge reinforces practical media literacy habits that can be applied in everyday life.

---

# Objective

Build a complete offline HTML application that teaches media literacy through interactive discovery, helping users recognize misleading headlines, emotional persuasion techniques, and manipulation strategies before accepting information as factual.

---

# Features Implemented

## Guided Learning Experience

The application begins with an interactive introduction explaining:

- What media literacy is
- Why misleading information spreads quickly
- How headlines influence perception
- Why emotional content is often more persuasive than factual content

Users are guided step-by-step before beginning each challenge.

# Challenge 1 – Headline Detective

The first challenge teaches users to distinguish between attention-grabbing headlines and factual reporting.

Users are presented with:

- A fictional news headline
- A matching article
- A realistic news layout

They are asked:

- Would you click this? (Yes / Maybe / No)
- Which parts of the headline appear exaggerated or misleading?

---

# Headline Analysis

After the user's response, the analyzer reveals:

- Headline Accuracy Score
- Source Reliability Score
- Highlighted misleading phrases
- Detailed explanation
- Headline overreach analysis
- Fair rewritten headline
- Practical takeaway

This helps users understand how sensational wording can distort otherwise ordinary news.

---

# Challenge 2 – Emotion Detector

The second challenge focuses on emotional persuasion.

Users analyze a fictional:

- Social media post
- News excerpt
- Reel caption

They identify:

- Their emotional reaction
- Words that triggered the reaction

---

# Emotional Manipulation Analysis

The analyzer then reveals:

- Emotional Manipulation Score
- Audience Targeting Score
- Intended emotional response
- Manipulation technique used
- Emotional trigger words
- Neutral rewritten version
- Learning takeaway

This demonstrates how language can influence opinions without changing the underlying facts.
