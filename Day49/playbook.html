What is this tool?

My AI Playbook is a personal, private toolkit for getting better, more consistent results out of AI models like Claude — built around how you actually work as a developer.

The main sections

  • Dashboard — a quick overview and jump-off point into your categories.
  • Workflows — reusable prompt templates for recurring tasks (like debugging a stack trace), with fill-in-the-blank variables so you're not typing from scratch each time.
  • Prompt Builder — build a prompt from labeled parts (role, objective, context, constraints, etc.) with a live preview, so you learn what makes a prompt effective.
  • Loop Builder — turn a prompt into a set of standing instructions the AI can follow repeatedly on its own, with clear stopping and safety rules.
  • Saved / Favorites — quick access to workflows you've starred or created yourself.

Where's my data?

Everything is stored in this browser's local storage only. Nothing is uploaded. Use Export to back it up or move it to another device.

Keyboard shortcuts

  • / — focus the workflow search box
  • N — create a new workflow
  • 1–5 — jump to Dashboard / Workflows / Prompt Builder / Loop Builder / Saved
  • Esc — close any open panel or modal
