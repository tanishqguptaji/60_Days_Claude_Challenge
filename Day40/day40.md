# Day 40 – Build Your Own AI Assistant

## ABTalks 60 Days Claude Challenge

## Project Overview

Today I built **Resume Audit**, an AI-powered assistant that performs recruiter-style resume screening and ATS analysis. The assistant is designed to help job seekers evaluate their resumes by identifying strengths, ATS compatibility, missing sections, potential red flags, and actionable improvements.

Unlike a traditional chatbot, this assistant provides structured hiring-style feedback, making the experience feel like interacting with an experienced recruiter rather than simply asking an AI questions.

---

# Objective

Build a production-ready AI assistant that:

- Interviews users through guided interactions
- Uses a specialized system prompt
- Produces recruiter-style structured outputs
- Features a purpose-built UI
- Delivers a polished AI product experience

---

# Assistant Overview

**Assistant Name**

Resume Audit

**Purpose**

Analyze resumes against recruiter expectations and ATS (Applicant Tracking System) standards.

**Primary Goal**

Help candidates understand:

- ATS readiness
- Resume strengths
- Missing information
- Recruiter concerns
- Recommended improvements

---

# Core Features

## Resume Input

Users can:

- Paste resume text
- Submit complete resume content
- Run an AI-powered audit

---

## ATS Analysis

The assistant evaluates:

- ATS compatibility
- Resume structure
- Readability
- Keyword usage
- Technical skills
- Formatting quality
- Missing recruiter information

---

## Strength Analysis

The assistant identifies positive aspects such as:

- Technical focus
- Relevant skills
- Academic achievements
- Industry alignment
- Clear specialization

---

## Gap Detection

The audit highlights important missing sections, including:

- Contact information
- Education details
- Work experience
- Projects
- Skills section
- Quantifiable achievements
- Supporting evidence for claims

---

## Recommended Fixes

Instead of only identifying problems, the assistant generates prioritized recommendations, such as:

- Add measurable project achievements
- Improve ATS-friendly formatting
- Create a structured skills section
- Strengthen the professional summary
- Include work experience or projects
- Improve education details
