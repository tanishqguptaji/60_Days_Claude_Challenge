# Day 39 – PDF Splitter & Merger

## ABTalks 60 Days Claude Challenge

## Project Overview

Today I built **PDF Splitter & Merger**, a premium browser-based PDF utility that enables users to split and merge PDF documents directly within the browser.

The application is designed with a modern, commercial-quality interface while ensuring complete privacy by processing documents locally on the user's device. No files are uploaded to external servers, making the tool fast, secure, and suitable for offline use after the initial load.

---

# Objective

Build a premium single-page PDF utility that allows users to:

- Split PDF documents into multiple files
- Merge multiple PDFs into one document
- Preview uploaded files
- Process everything locally
- Deliver a smooth and responsive user experience

---

# Features Implemented

## PDF Splitter

The Split tool allows users to upload a PDF and prepare it for extraction.

Supported capabilities include:

- Upload PDF files
- Detect total number of pages
- Display page previews
- Split by custom page ranges
- Split after selected pages
- Split every N pages
- Extract selected pages
- Validate page ranges before processing
- Preview output structure before splitting

The application highlights invalid page ranges and provides user-friendly validation messages.

---

## PDF Merger

The Merge tool enables users to combine multiple PDF files into a single document.

Features include:

- Drag-and-drop file upload
- Multiple PDF selection
- Sortable file list
- File reordering before merge
- File previews
- Page count detection
- Estimated output size
- One-click PDF generation
- Download merged document

Users can easily rearrange the merge order before processing.

---

# Local Processing

All document operations are performed entirely inside the browser.

Benefits include:

- No server uploads
- Better privacy
- Faster processing
- Offline functionality
- Secure document handling

The application ensures that user files never leave their device.
