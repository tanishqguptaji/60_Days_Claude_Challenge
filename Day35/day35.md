# Day 35 – Prompt Puzzle: Master AI Prompting Through Play

## ABTalks 60 Days Claude Challenge

## Project Overview

Today I built **🧩 Prompt Puzzle**, an interactive game-based learning application that teaches prompt engineering through hands-on challenges instead of passive reading.

Rather than simply explaining prompt engineering concepts, the application places users inside realistic AI prompting scenarios where they build prompts, clean poorly written prompts, compare multiple prompts, and learn why some prompts consistently produce better AI outputs.

The experience feels like a puzzle game while secretly teaching professional prompt engineering principles.

Build a premium single-file HTML application that helps users master AI prompting through interactive gameplay, instant feedback, and progressive learning.

# Three Interactive Challenge Types

### 1. Build the Prompt

Users receive:

- Desired AI output
- Multiple prompt blocks

Their task is to drag only the correct prompt components into the prompt builder while ignoring misleading distractors.

This teaches users how well-structured prompts are constructed.

---

### 2. Clean the Prompt

Users are shown an over-engineered prompt filled with unnecessary complexity.

Their objective is to simplify it while preserving:

- Role
- Task
- Constraints
- Output format

This demonstrates that better prompts are usually clearer—not longer.

---

### 3. Choose the Best Prompt

Users compare multiple prompts designed to solve the same problem.

They must identify which prompt is most likely to produce the highest-quality AI response.

The application explains why one prompt performs better than the others.
