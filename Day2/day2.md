# Day 2 – Prompt Engineering

## What I Learned

Today I learned about Prompt Engineering, which is the skill of writing effective prompts for AI tools like Claude and ChatGPT.

Prompt Engineering helps in:

* Getting more accurate responses
* Reducing hallucinations
* Improving creativity
* Generating professional-level outputs
* Saving time by reducing repeated corrections

I understood that AI performs much better when prompts include:

* Clear instructions
* Context
* Role assignment
* Desired output format
* Specific requirements

  Weak Prompt → Basic Output

1. Explanation

## Output Observation

The engineered prompt produced a much more professional and detailed output with:

* Better explanations
* Structured formatting
* Cleaner design ideas

* More useful examples

* Prompt Engineering is an important AI skill.
