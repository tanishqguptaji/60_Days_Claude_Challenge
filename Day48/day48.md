# Project

## Compare & Decide Builder – Everyday Laptop Decider

Today I built an AI-powered decision intelligence application that helps users compare multiple laptop options using real-world research, transparent citations, and customizable weighted scoring.

Unlike traditional comparison websites, this tool explains *how* every recommendation is produced and clearly highlights conflicts or estimated values instead of hiding them.

---

# Features

- Interactive weighted comparison engine
- Live ranking updates
- Real research-backed comparison
- Transparent citation panel
- Data matrix with sourced specifications
- Conflict detection for inconsistent benchmarks
- Estimate flags for non-directly sourced values
- Responsive premium UI
- Single-file HTML application
- Built using HTML, CSS & Vanilla JavaScript
