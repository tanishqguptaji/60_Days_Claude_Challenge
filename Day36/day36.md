# Day 36 – Cognitive Pattern Explorer

## ABTalks 60 Days Claude Challenge

## Project Overview

Today I built **Cognitive Pattern Explorer**, a psychology-inspired interactive learning application that encourages users to reflect on how they naturally think, prioritize information, and make decisions.

Instead of treating psychology like a quiz or personality test, the application creates a calm, game-like experience where users explore their own cognitive tendencies through interactive scenarios, drag-and-drop activities, and reflective journaling.

The experience focuses on **self-awareness**, not diagnosis. Every insight uses reflective language to help users better understand themselves without placing them into rigid categories.

---

# Objective

Build a responsive, single-file offline web application that helps users discover their thinking patterns through interactive exercises while emphasizing reflection over evaluation.

---

# Features Implemented

## Calm & Stress Mode

Users begin by selecting their preferred exploration mode:

- 🌿 Calm Mode
- 🔥 Stress Mode

The interface adapts visually to create a different emotional atmosphere before beginning the journey.

---

# Chapter 1 – Discover Your Thinking Style

Users explore realistic everyday situations and choose responses that best reflect how they naturally think.

The experience introduces five thinking tendencies:

- Analytical Thinker
- Emotional Intuitive
- Overthinking Loop Style
- Action-First Decision Maker
- Balanced Reflective Thinker

Rather than assigning labels, the application gradually builds a personalized reflection profile.

---

# Chapter 2 – Choose Your Priorities

Users rank what matters most when making decisions using an interactive drag-and-drop interface.

Examples include:

- Getting the facts right
- Trusting intuition
- Moving quickly
- Avoiding mistakes
- Considering everyone's feelings

This activity helps reveal personal decision-making priorities.

---

# Chapter 3 – Map Your Thinking

Users organize thoughts into the order they naturally occur during a real-life situation.

Using drag-and-drop timelines, the application visualizes how people typically process:

- Emotions
- Analysis
- Reflection
- Action
- Reconsideration

This demonstrates that different people naturally follow different thinking sequences.

---

# Personalized Reflection Journal

After completing all chapters, users receive:

- Personalized insight summary
- Percentage breakdown of thinking tendencies
- Animated visualization
- Reflection journal
- Local journal saving

Instead of giving a score, the application encourages self-reflection and continuous growth.
